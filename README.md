### 介绍 ⭐
Hey, this is wowking. 

我把自己平时所学所思所感整理进了这个博客里面，希望这个博客中的内容能帮助到大家，搭建内容修改自 [Hux](https://github.com/Huxpro/huxpro.github.io)，具体过程参考了 [qiubaiying](https://github.com/qiubaiying/qiubaiying.github.io). 
 
更为详细的教程戳这 [wiki](https://github.com/qiubaiying/qiubaiying.github.io/wiki/%E5%8D%9A%E5%AE%A2%E6%90%AD%E5%BB%BA%E8%AF%A6%E7%BB%86%E6%95%99%E7%A8%8B)

### [查看博客戳这里 👆](https://wowking2018.github.io/) 

博客最初的想法来自 `CLQ`，感谢他的分享，但是原环境并不支持对于数学公式的写作，故我们添加了Mathjax的环境对公式进行渲染，后续还想增加更多方便的内容. 对我个人来说，博客会成为一个记录自己成长历程的平台，我希望多年以后我回看这一切的时候，会感到欣慰与感慨.
